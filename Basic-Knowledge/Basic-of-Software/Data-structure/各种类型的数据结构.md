## 树  Tree
* 二叉树

> * 一个根节点开始为root

> * 再次插入数据的时候，比根节点小放左边，大就放右边

> * 任一一个节点只能有一个父节点，子节点

> * 从root开始，深度为1，多经历一个子节点，深度+1
