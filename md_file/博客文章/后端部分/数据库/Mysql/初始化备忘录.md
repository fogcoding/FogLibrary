## 初始化备忘录



#### 设置时区为东八区

```sql
show variables like '%time_zone%';
select now();
set global time_zone = '+8:00';
flush privileges;
```

#### Windows下安装

```mysql
-- 安装服务并制定配置文件
mysqld install mysql --defaults-file=d:\datacore\xxxxxxx
-- 初始化基础文件
mysqld --intialize
-- 启动服务
net start mysql
-- 修改密码
-- https://www.cnblogs.com/mrhonest/p/10881646.html

```
